## GFP-OpenWorld

OpenGL 后室游戏 —— 浙江大学《计算机图形学》期末大作业

游戏简介：你出生在了后室。在躲避实体的同时找到前来帮助你的米塔，并使用她给你的武器消灭实体，逃出这个地方。

使用 Visual Studio 2022 进行开发。
